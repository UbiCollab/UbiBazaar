Getting started

Register new user

To start using this product the first step is to register an account on the website. After you have registered an account you need to activate it. Depending on the settings set by the site administrator you will either get a confirmation mail sent to the e-mail address you entered, or confirmation is done manually by an administrator. If activation is done by an administrator you will receive a notification mail when the account has been activated. Activation by clicking a link in an e-mail is the most common setting.

 

 

 

 

Logging in is now possible, and the following picture shows the main screen which is the first you see after logging in. Here you see a newsfeed of updates about applications you have downloaded or are following, as well as updates about users you are following. The newsfeed may not be as populated as the one you see in this picture, but as you follow more friends and applications more updates will be shown.

 

 

 

 

Register a new device

The next step you might consider is to add one of your devices like a smart phone or a tablet device. This is done by going to the “My Stuff”-tab, select the “Device”-tab. and click the blue “New device?” link. The link will send you to a form for registering a device, the information you need is  a proper name, and your device IMEI number which can be found by dialing *#06# on your device. If you have downloaded the UbiHome client for Android and logged in the device is automatically registered to the UbiHome account you signed in with.

 

 

 

 

Add an application project

To add an application you need to select the “My Stuff”-tab, click the “Apps”-tab, and then click the blue link “Create new project”. Enter the necessary information and submit. Now you are redirected to the “update” project tab where you can upload an application file. Here you need to fill in version number, a description, what platform the application file supports and if this application is public or not. See section “Application management” if you are unsure if you should chose to upload a public or private application-file.

 

 

 

 

 

Application Management

 

This is the overview you get of an application project you have released. Here you can add various information. You can add screenshots and icons, manage different releases and view submitted bugs, make groups for specially invited users and make different forums, either for public use, or just for a group. The “update” link makes it possible to upload new versions of your application.

If you only want specific users to access you application you should leave the “Public” checkbox unchecked, and invite users to a group. The users you invite can then find your application under “My Stuff” – “Groups”.

 

 

 

 

 

Browse and download an application

This is the view you see when you go to “Browse Apps” and click an application. From here you can download the application to your device by using the bar code-scanner on your device. You have to install the UbiHome client first to use the other bar codes on the website.

To be able to give feedback, report bugs or add tags you need to have that application installed. Report issue can be used if you think the application has a virus. A message is then sent for evaluation by an administrator.

 

Social Features

Compose a private message

This is the view you get when you want to compose and send a private message.

When you enter the receiving user’s name, a drop down-list will appear to help you select the correct one. Then, after filling in the ‘subject’ and the ‘message’ you want to send, simply press the submit button to send the message. The message will be saved in your “outbox” folder.

It is also possible to compose a message when visiting a user-profile. Press the button “send message” to get to the same form as shown under.

Following users and applications

While browsing applications and user profiles, you might find an application you are interested in receiving updates and status messages about or you might be interested in a certain developers work. To start receiving relevant information regarding an application or a specific user, you may start “following” them. To do so browse to the user profile you are interested in and click the “Follow”-button below the profile picture, as seen in the image below. If you are already a follower you will have the possibility to “unfollow” instead.

To follow a specific application, browse to the relevant application and click the follow-button on the bottom of the Information-tab.

 

Groups

If you are the owner of an application you might be interested in creating one or more groups. For instance; you might want to create a group for users you trust so that they can test your application. If you set the group to be private, only users you invite may join the group. This ensures that the feedback and message posts you receive through the group are from people you trust. As the owner of a group you can assign different roles to the group members depending on what you want them to be able to do.

Message board

If you are a member of a group surrounding an application, you can access the message board. The message board is a forum where the users can discuss topics related to an application. The owner of the app and a group administrator has the ability to ‘stick” a topic, so that it always will appear on top of the list. To create a new topic, simply press the ‘new thread’ button.

 

Settings

 

To change your profile settings you need to go to the “My Stuff”-tab, and select the “Settings”-tab. Here you can add a profile picture, change information about yourself, and change your password if needed.
